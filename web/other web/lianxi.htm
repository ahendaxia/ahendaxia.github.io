广州  — 元音古寺的乘车线路

  • 从广东火车总站坐公交车257路 → 天河客运站（因客运公司班车变动，建议预备乘此路线者请先联系客运站确认班次，天河客运站电话：020-37085070），后在天河客运站乘坐广州 → 龙门（或紫金）班车；
  • 坐车时记得询问司机，车辆过响水镇后，在白芒林场派出所路口下车，路边有元音古寺的指路牌；
  • 下车后还有4公里路到元音古寺，可在近处询问摩托车载您进寺，或步行进寺。

惠州  — 元音古寺的乘车线路

  • 可于惠州汽车站乘坐班车到博罗汽车站，后在博罗汽车站乘坐至柏塘方向的班车（买元音古寺的票），下车情况如上。

深圳  — 元音古寺的乘车线路

  • 线路1：可于东湖汽车站（因客运公司班车变动，建议预备乘此路线者请先联系客运站确认班次，东湖汽车站咨询电话：0755-25006060）乘坐深圳 → 柏塘班车，在白芒林场派出所下车，下车情况如上。
  • 线路2：深圳罗湖火车站下车，可于附近数百米的长途汽车站（因客运公司班车变动，建议预备乘此路线者请先联系客运站确认班次，长途汽车站电话：0755-82321670）乘坐深圳 → 博罗班车，下车情况如上。
  • 线路3：可于布吉联检站乘坐至河源方向的班车（随车联系电话：13715399610）到十二岭下车，坐到博罗方向的班车到白芒林场派出所下车，下车情况如上。
